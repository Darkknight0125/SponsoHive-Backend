# Testing Guidelines

## Prerequisites
- Node.js (v18+ recommended)
- NPM

## Setup Instructions

1. Clone the repository:
   bash
   git clone https://github.com/opencodeiiita/SponsoHive-Backend.git
   cd SponsoHive-Backend

2. Install dependencies:

    npm install

3. Run tests locally:

    npm test

4. Run tests in watch mode for live feedback:

    npm run test:watch
